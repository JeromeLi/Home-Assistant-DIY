## Add-on Samba的安装、配置与使用

samba是什么？

实现网络文件共享的工具软件。

*注：samba启动前，需要配置好用户名和密码*

## 共享文件夹介绍

|  文件夹   | 内容  |
|  ----  | ----  |
| config  | HomeAssistant的配置目录 |
| backup  | 存放系统备份内容 |
| ssl  | 存放加密传输用的私钥与证书文件 |
| media | 媒体文件 |
| addons  | 存放本地add-on，一般用于add-on开发 |
| share  | 无特定目的的共享内容 |


## Add-on是什么

- Add-on是与HomeAssistant并行运行的其它软件
- Add-on可以通过HomeAssistant Supervisor安装/启动/配置
- Add-on一般与HomeAssistant有关联
